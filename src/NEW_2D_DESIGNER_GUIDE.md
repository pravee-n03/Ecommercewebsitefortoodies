## 📋 **HOW TO USE (CUSTOMER)**

### **Step 1: Admin Configures Product**
Admin must first:
1. Upload product in Admin → Products
2. Configure 2D model in Admin → 2D Models
3. Upload **mockup image** (t-shirt template)
4. Set printing methods with costs

### **Step 2: Customer Opens Designer**
1. Go to Shop
2. Find product with "Create 2D Design" button
3. Click button → Designer opens

### **Step 3: Customer Uploads Design**
1. **Select printing method** from dropdown:
   - Screen Print: ₹15/sq.in (Min: ₹100)
   - DTG Print: ₹25/sq.in (Min: ₹150)
   - Vinyl Print: ₹20/sq.in (Min: ₹120)

2. **Click "Choose Image File"** button
3. **Select your design** (logo, artwork, photo)
4. Image appears on canvas over mockup
5. **Drag to position** it
6. **Resize** using corner handles
7. **Rotate** using rotation handle

### **Step 4: Adjust Design**
- Use zoom controls to see details
- Toggle grid to see print area boundaries
- Toggle mockup to see just your design
- Add multiple layers by uploading more images

### **Step 5: Save**
1. Watch cost calculate in real-time
2. Click **"Save Design"** button
3. Design saves to Studio tab
4. Navigate to Studio → Purchase it!

## 🔧 **TROUBLESHOOTING**

### Problem: Designer won't open
**Solution:** Admin hasn't configured model for that product

### Problem: Can't upload design
**Solution:** Select a printing method FIRST

### Problem: Mockup doesn't show
**Solution:** Admin forgot to upload mockup image in model config

### Problem: Cost shows ₹0.00
**Solution:** Design is too small or printing method not selected
