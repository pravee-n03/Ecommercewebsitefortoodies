## 🆘 Troubleshooting

### "Supabase project is paused"
**Fix:**
1. Go to Supabase dashboard
2. Click **Resume project**
3. Wait 30 seconds
4. Run SQL again

### "Permission denied"
**Fix:**
1. Verify you're logged into Supabase
2. Check you're on the correct project
3. You need Owner/Admin access

### "Syntax error in SQL"
**Fix:**
1. Make sure you copied the **entire** SQL file
2. Check for extra characters at start/end
3. Re-copy from `/database/fresh-setup-v2.sql`

### "Error persists after running SQL"
**Fix:**
1. Hard refresh: Ctrl+Shift+R (or Cmd+Shift+R)
2. Clear browser cache
3. Try incognito/private window
4. Check browser console (F12) for specific errors
